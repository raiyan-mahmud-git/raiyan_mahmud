# Raiyan Mahmud — Portfolio

This is the personal portfolio website of me, a prospective Computer Science student focused on machine learning, data analysis, and Python development. The site showcases skills, projects, education, certifications, and contact information.

## Features
- Responsive design for desktop, tablet, and mobile
- Hero section with profile image, intro, and call-to-action buttons
- Skills showcase with hover effects
- Projects gallery with tech stack labels and GitHub links
- Education and certifications sections
- Contact section with clickable links
- Interactive elements: hamburger menu, smooth scroll, animated cursor glow
- Modern dark theme with accent colors and subtle animations

## Technologies Used
- HTML5
- CSS3 (Grid, Flexbox, Animations)
- Google Fonts
- JavaScript (for hamburger toggle, scroll effects, cursor glow)

## Usage
1. Clone the repository: `git clone <repository-url>`
2. Open `index.html` in a web browser
3. Ensure images are in the `/images` folder with correct names

## Customization
- Replace `profile.jpg` with your own image
- Update skills, projects, education, and contact links in `index.html`
- Adjust accent colors and fonts in CSS to match your branding
